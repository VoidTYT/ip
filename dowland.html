Установка Python

Установка Python происходит с официального сайта. На главной странице которого вы увидите большую желтую кнопку, нажав на которую, вы сможете загрузить самую последнюю версию Python. На данным момент это Python 3.10.

После нажатия на кнопку, у вас начнется обычная загрузка exe-файла. Скачиваем его.

Открываем. Нажимаем запустить.

В открывшемся окне ОБЯЗАТЕЛЬНО СТАВИМ ГАЛОЧКУ возле слов "Add Python 3.10(или другой версии, смотря какая у вас) to PATH"

Затем нажимаем "Install Now". После чего начнется загрузка Python на ваш компьютер.
